## Key Metrics

- Time-to-First-Useful-Result
- Trust Calibration Accuracy
- Fallback-to-Human Escalation Rate
- User Satisfaction Score

## Best Practices
- Design hybrid interfaces, never pure chat — half the population has limited writing proficiency
- Stream output to reduce perceived latency (5s wait becomes engaging progressive reveal)
- Communicate uncertainty explicitly — show confidence levels, source attributions
- Design for iterative refinement, not one-shot perfection
- Implement graceful degradation with human fallback for out-of-scope queries
