## Key Metrics
- Cost per Query (by model tier)

- Model Routing Distribution
- Token Waste Ratio
- Cost-Quality Pareto Position
- Monthly API Spend Trend

## Best Practices
- Implement prompt caching first — 75% discount on cached tokens, highest ROI
- Route by intent — simple queries to cheaper models, complex to capable (40-60% savings)
- Measure and optimize token consumption daily
- Use semantic caching cautiously — bad matching returns wrong cached results
- Model cost against quality on a Pareto curve for every use case
